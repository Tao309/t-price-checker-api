# Backend часть плагина для браузера. Мониторинга цен на товары в корзинах маркетплейсов

## Реализовано api:
- привязка/отвязка источника товара
- привязка/отвязка источника книги
- получение книг по названию
- получение источников товаров по названию
- изменение товара в архив и обратно
- удаление стока
- получение товаров по типу маркетплейса
- сохранение книги
- сохранение источника товара
- сохранение одного и нескольких товара
- создание сущности "Издательский дом"
- создание сущности "Издательский бренд"
- создание сущности "Издательство"
